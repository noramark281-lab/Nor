# MEXC Event Trader - العقود الآجلة للأحداث

تطبيق Flutter لتداول العقود الآجلة للأحداث على منصة MEXC.

## الميزات

- 🔐 أمان API مع تشفير AES وتحقق بيومتري
- 🤖 بوت تداول تلقائي مع 7 استراتيجيات
- 📊 شموع يابانية تفاعلية
- ⏱️ فترات زمنية متعددة (10 دقيقة إلى 1 يوم)
- 🔔 إشعارات فورية
- 🌙 وضع داكن/فاتح

## البناء

```bash
flutter pub get
flutter build apk --release
```

## متطلبات النظام

- Android 5.0+ (API 21+)
- Flutter 3.24+

## الأمان

- مفاتيح API مشفرة بـ AES-256-CBC
- مصادقة بيومترية (بصمة/وجه)
- Keystore أصلي Android
