# Nor - أقوى تطبيق تداول حقيقي لعقود الأحداث على MEXC 🚀

تطبيق Nor هو حل احترافي متكامل للتداول الآلي واليدوي على منصة MEXC، مصمم ليكون الأسرع والأكثر أماناً وموثوقية في السوق الحقيقي.

## 🌟 الميزات الأسطورية
- **تحديثات لحظية (WebSockets)**: تتبع الأسعار بجزء من الثانية دون أي تأخير، مما يمنحك أفضلية في دخول الصفقات.
- **تداول حقيقي 100%**: مرتبط مباشرة بـ API MEXC لتنفيذ الأوامر الحقيقية ومزامنة الأرصدة.
- **بوت ذكاء اصطناعي متطور**: استراتيجيات تداول مبنية على مؤشرات فنية دقيقة (RSI, Bollinger Bands, SMA Crossover).
- **إدارة مخاطر احترافية**: نظام حماية مدمج يمنع الخسائر الكبيرة عبر تحديد حجم الصفقة وإيقاف البوت تلقائياً عند الخسائر المتتالية.
- **تخزين دائم (SQLite)**: سجل تداول كامل محفوظ على جهازك، لا تضيع بياناتك أبداً حتى بعد إغلاق التطبيق.
- **أمان فائق**: تشفير كامل للبيانات الحساسة واستخدام GitHub Secrets في عملية البناء.

## 🛠 التقنيات المستخدمة
- **Flutter**: لبناء واجهة مستخدم عالمية وسريعة.
- **WebSockets**: للاتصال المباشر واللحظي مع البورصة.
- **SQLite**: لإدارة قواعد البيانات المحلية.
- **Provider**: لإدارة حالة التطبيق بكفاءة عالية.
- **GitHub Actions**: للبناء التلقائي وإصدار نسخ APK آمنة.
